🏥 Hospital Management System (Java Swing)
📌 Project Overview

The Hospital Management System is a desktop-based application developed using Java Swing. It helps manage hospital operations such as patient records, doctor details, and appointments through a simple and user-friendly graphical interface.

💻 How to Run the Project
Step 1: Install Requirements
Install Java JDK (8 or above)
Install VS Code / IntelliJ / Eclipse
Step 2: Run the Code

Copy the code into a file named:

HospitalManagementSystem.java
Open terminal in project folder

Compile:

javac HospitalManagementSystem.java

Run:

java HospitalManagementSystem
